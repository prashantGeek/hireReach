# Linkedin Outreach Workspace

This repository contains two separate Node projects:

- [frontend](frontend) — Next.js app
- [backend](backend) — Node/Express API

## Run the apps separately

Frontend:

```bash
cd frontend
npm run dev
```

Backend:

```bash
cd backend
npm run dev
```

## Notes

- There is no root `package.json` or root lockfile.
- Install dependencies only inside the app folder you are working on.
- Keep frontend and backend dependency trees isolated.
